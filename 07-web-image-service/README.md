# Web Service for CV

## Dependencies
```bash
# Python
pip3 install flask flask-cors
pip3 install opencv-contrib-python
pip3 install numpy

# Node
npm install -g http-server
```
## How to run
```bash
cd Client && http-server . &
cd ../Server && python3 run.py
```

go to `localhost:8080` on your browser.  
Have fun.
